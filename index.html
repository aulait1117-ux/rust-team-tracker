ルームに接続

チーム・メンバーのリストをリアルタイムで共有するための「ルーム」を作成するか、 友達から教えてもらったルームコードで参加できる。コードを知っている人だけが同じデータを見られる。

ログイン傾向（時間帯別）

使い方

1. ルーム接続

  • 初回は「🆕 新しいルームを作る」でランダムなルームコードを生成するか、コードを入力して「参加する」で既存の部屋に参加する。
  • 一度接続すれば次回以降そのブラウザは自動で同じルームに再接続される。
  • 画面上部の「ルームコード」を「📋 コピー」して友達に伝えるだけで共有できる。「🔄 別のルームに切り替える」で別の部屋に移動できる。

2. チーム登録

  • 「＋ チーム追加」でチームを登録。種別は「⚠ 敵」（赤）／「🟢 味方」（緑）から選べ、カード左端の色でも見分けられる。チーム名の横のボタンをクリックすると後から種別を切り替えられる。

3. メンバー追加

  • 「名前」欄に2文字以上入力すると名前サジェストが出る。候補をクリックするとBattleMetrics IDが自動で紐付き、相手が改名しても追跡できるようになる。
  • 手入力してそのまま「＋ メンバー追加」もできる（この場合IDは未紐付け）。
  • 同名が多くて区別できないときは、隣の「Steam ID / プロフィールURL」欄で追加する。Steam ID（17桁前後の数字）でも、プロフィールのURL（https://steamcommunity.com/profiles/… や …/id/…）を貼り付けてもよい。URLからIDを自動で取り出す。名前を空にしてSteamだけで追加すると、Steamの表示名を仮の名前にする。
  • 同じSteam IDは二重登録されない（名前が同じでもSteam IDが違えば別人として追加できる）。
  • 「複数名を改行区切りで貼り付け」→「＋ 一括追加」で複数人まとめて登録できる。
  • 「部分一致」にチェックを入れると、名前の一部が含まれているだけでオンライン判定される。
  • 後から「🔗 ID紐付け」ボタンでも、手入力済みのメンバーにBattleMetrics IDを紐付けられる。

4. オンライン確認

  • 30秒ごとに自動でオンライン状況を更新する。「🔄 今すぐ確認」で手動更新もできる。
  • OFFのメンバーには「最終IN: HH:MM」が表示される。
  • 誰かがOFF→ONになった瞬間、トースト通知・通知音（「🔊 通知音：ON/OFF」で切替）・デスクトップ通知（「🔔 デスクトップ通知を有効化」で許可）で知らせる。

5. 拠点写真

  • 各チームカード内の貼り付け欄をクリックして選択し、Ctrl+Vでスクリーンショットをそのまま貼り付けられる（ファイル選択でも追加可）。
  • サムネイルをクリックすると拡大表示、×ボタンで削除できる。
  • 貼り付けた写真はルーム接続中、同じルームの全員に共有される（クラウド同期される）。共有したくない画像は貼り付けないこと。

6. 友達との共有

  • ルームコードを伝えるのが基本の共有方法。
  • このindex.htmlファイル自体を友達に送ってもよい。file://で直接開いても、そこでルームコードを入力すればFirebase経由で自動的に同じデータに同期される。

7. バックアップ

  • 「📤 共有用リンクを作成」「📋 テキストで共有」でチーム構成をURL・テキストとしてバックアップ・共有できる（写真は含まれない）。
  • 「📥 テキストから読み込む」でその内容を読み込んでマージできる。
